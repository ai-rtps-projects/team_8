# 🍔 AI Food Ordering Chatbot (Campus Bite)

## 📌 Project Title

**AI-Powered Online Food Ordering Assistant Chatbot**

---

## 🎯 Aim

To design and develop an **AI-based food ordering chatbot** using Flask that allows users to browse menu items, manage cart, and place orders via chat.

---

## 📝 Description

This project is an intelligent chatbot integrated into a food ordering system.
Instead of manually browsing menus, users can interact conversationally.

Example:

* "Show menu"
* "Add burger"
* "Checkout"

---

## 🎯 Objectives

* Simplify food ordering
* Provide conversational UI
* Enable smart cart management
* Improve user experience

---

## ⚙️ Working Principle

The chatbot uses **rule-based NLP (keyword matching)**.

### Flow:

1. User sends message
2. Backend processes input
3. Intent detected:

   * Menu → Show items
   * Add → Add to cart
   * Cart → View cart
   * Checkout → Place order
4. Response returned via API
5. UI displays response

---

## 🧠 Algorithm

1. Start Flask server
2. Load menu dataset
3. Accept user input
4. Normalize text
5. Detect intent
6. Perform action
7. Return response
8. Repeat
